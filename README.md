## model requirements

- vaststream：编译器和推理引擎
- vamc：模型转换工具

## model summary

按算法方向主要从以下两个方面总结，该系列主要用来展示目前支持和将要支持的模型:

- arch summary : 该算法方向的整体发展脉络和重点模型
- op&module summary：该算法方向的算子清单和通用结构
